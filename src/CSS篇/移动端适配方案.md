假如我们让5rem 等于屏幕的50%

我们知道

```javascript
  var Root = document.documentElement;
  
  1rem  =  Root.style.fontSize + 'px'
```

所以我们可以设置HTML根节点的fontSize为屏幕的宽度除以5 * 2



```javascript
  var Root = document.documentElement;
  Root.style.fontSize = window.innerWidth / 10 + 'px';
```



如此，在任何屏幕下5rem均是屏幕的50%
